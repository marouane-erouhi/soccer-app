# Scoccer app
A webapp for managing soccer players and edeting their properties built using Node.js and Express.js for the backend and React.js and MaterializeCss for the frontend
## Picture

## Features
### Backend
- View players
- Add players
- Delete players
- Update player data
### Frontend
- View players
- Add new players
- Delete players
## Technologies
- Node.js
- Express.js
- React.js
- Axios
- MongoDB with Mongoose
- MaterializeCss
- Bable
- Nodemon
